    1.vim xxx.jar 搜索修改，同Linux修改zip;  
    2.提取修改再覆盖；jar xf / jar uf  
    3.解压jar包，修改后重新打包

    1.http://IP:端口/服务/请求映射
    2.端口：Tomcat默认端口，server.xml配置；服务名：webapps中的服务名称
