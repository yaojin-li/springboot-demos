# springboot-apollo

### 笔记
1.Idea 中启动，配置启动参数：
注意：快速构建quick-start官方教程的默认端口为8070
-Dapollo.configService=http://172.190.119.83:8080 -Denv=DEV

2.java命令启动：
java -Dapollo.configService=http://172.190.119.83:8080 -Denv=DEV -jar apollo-demo.jar
